# mcp-server-example — context for AI coding agents

This is a **standalone MCP server** scaffolded with EAP-Core.

- **No LLM client here** — this project just *exposes* tools. Other
  agents call into them via the MCP protocol over stdio.
- **Add a tool**: ``eap create-tool --name <name> --mcp`` from this
  project's root. The decorator generates JSON Schema from type hints;
  wire the tool into the explicit ``McpToolRegistry`` in ``server.py``.
- **Authoritative interface contract**: the JSON Schema generated by
  ``@mcp_tool`` IS the MCP contract callers see. Keep type hints
  narrow and accurate.
- **Run**: ``python server.py`` (the server speaks MCP over stdio).
- **Connect** from a Claude Code agent or any MCP-aware client by
  pointing it at ``python server.py`` as the command.

For full SDK design see ``docs/superpowers/specs/2026-05-10-eap-core-design.md``.
