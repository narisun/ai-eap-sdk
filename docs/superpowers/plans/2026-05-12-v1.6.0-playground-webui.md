# v1.6.0 Implementation Plan — Playground web UI for example agents

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Ship a `examples/playground/` project — a minimal FastAPI + vanilla-HTML/JS web UI that lets SDK users interact with the existing example agents via browser. Auto-discovers any `examples/*/agent.py` that exports `build_client()` and exposes them in a dropdown. The headline differentiator vs a generic chat UI: a **tool-call trace panel** showing which middleware fired, which tool got invoked, and what the args were — a debugging/learning aid for SDK users.

**Architecture:** Single-page application served by FastAPI. Backend: ~120 lines that scan `examples/*/agent.py`, import each via `importlib.util`, cache `build_client()` results, and expose chat + tools endpoints. Tracing: a small `PlaygroundTraceMiddleware` (purely additive to the SDK; lives in the playground project, not in `eap_core.middleware`) records tool invocations to a per-request list on `ctx.metadata`. Frontend: one HTML file + one JS file + one CSS file, no framework. Layout: agent dropdown at top, chat history + input center, tool-list sidebar + trace panel right.

**Tech Stack:** Python 3.11+, FastAPI (already in `[a2a]` extra; the playground project lists it as a direct dep), uvicorn for the server, vanilla HTML/JS/CSS. No new dependencies on `eap-core` itself.

**Scope:**
- IN: agent auto-discovery; chat endpoint (POST message → response + tool trace); tools-list endpoint per agent; manual tool-invocation endpoint (test tools directly); single-page UI with chat + sidebar + trace panel; README; integration test that spawns the server and exercises the API.
- OUT (deferred to future minor): streaming responses (SSE/WebSocket); auth on the playground itself (it's localhost-only); persistence of chat history; multi-turn conversation context (each message is independent — agents that need conversation state can wire it themselves); deploying to anything beyond localhost.

**Backward compat:** Strict additive. No SDK changes. New `examples/playground/` project.

- [ ] **Step 1.2: `tracing.py` — `PlaygroundTraceMiddleware`**

Records tool invocations to a per-request list on `ctx.metadata['playground.trace']`.

```python
"""Per-request tool-call tracing middleware for the playground UI.

Records every ``on_tool_call`` event into a list stored on
``ctx.metadata['playground.trace']``. The list is initialised on
``on_request`` and read by the playground server after the agent's
``generate_text`` / ``invoke_tool`` returns, then shipped to the
frontend's trace panel.

This middleware is **playground-local** — it's NOT in
``eap_core.middleware``. SDK users who want production tracing should
use ``ObservabilityMiddleware`` (which emits OTel spans). This is a
debugging/visualisation aid scoped to the playground.
"""

from __future__ import annotations

import time
from typing import Any

from eap_core.middleware.base import PassthroughMiddleware
from eap_core.types import Context, Request, Response


class PlaygroundTraceMiddleware(PassthroughMiddleware):
    """Records tool invocations + middleware entry/exit for the
    playground's trace panel.

    Each entry is a dict: {kind, name, args?, result?, duration_ms?, ts}.
    Frontend renders them as a timeline.
    """

    name = "playground_trace"

    async def on_request(self, req: Request, ctx: Context) -> Request:
        ctx.metadata["playground.trace"] = []
        ctx.metadata["playground.trace_start"] = time.perf_counter()
        return req

    async def on_tool_call(
        self, tool_name: str, arguments: dict[str, Any], ctx: Context
    ) -> dict[str, Any]:
        trace = ctx.metadata.get("playground.trace")
        if trace is not None:
            trace.append({
                "kind": "tool_call",
                "name": tool_name,
                "args": _make_json_safe(arguments),
                "ts_ms": (time.perf_counter() - ctx.metadata.get(
                    "playground.trace_start", time.perf_counter()
                )) * 1000,
            })
        return arguments

    async def on_response(self, resp: Response, ctx: Context) -> Response:
        trace = ctx.metadata.get("playground.trace")
        if trace is not None:
            trace.append({
                "kind": "response",
                "ts_ms": (time.perf_counter() - ctx.metadata.get(
                    "playground.trace_start", time.perf_counter()
                )) * 1000,
            })
        return resp


def _make_json_safe(value: Any) -> Any:
    """Convert non-JSON-serialisable values to strings. The frontend
    receives JSON; pydantic models / Paths / etc. become their repr.
    """
    import json
    try:
        json.dumps(value)
        return value
    except (TypeError, ValueError):
        return str(value)
```

Note: verify the actual `Middleware` Protocol signature — `on_tool_call` may not be a hook name the SDK exposes. Check `packages/eap-core/src/eap_core/middleware/base.py` first. If `on_tool_call` isn't a real hook, fall back to hooking only on_request/on_response and find another way to capture tool calls (perhaps wrapping the tool registry's `invoke` for the playground session).

- [ ] **Step 1.3: `server.py` — agent discovery + API**

```python
"""EAP-Core playground server — browser UI for example agents.

Discovers every ``examples/*/agent.py`` that exports ``build_client()``,
mounts a single-page UI at ``/``, and exposes a JSON API:

- ``GET  /api/agents``                 — list discovered agents + their tool names
- ``POST /api/agents/{name}/chat``     — send a message; returns response + trace
- ``POST /api/agents/{name}/tools/{tool}`` — invoke a specific tool directly

Discovery scans the playground's parent directory (``examples/``) for
subdirectories containing an ``agent.py`` whose module exports
``build_client``. Each agent is imported lazily on first reference and
the resulting ``EnterpriseLLM`` cached for the server lifetime.
"""

from __future__ import annotations

import importlib.util
import sys
import traceback
from pathlib import Path
from typing import Any

from fastapi import FastAPI, HTTPException
from fastapi.responses import FileResponse, JSONResponse
from fastapi.staticfiles import StaticFiles
from pydantic import BaseModel

# Module-level registry of {agent_name: build_client() result}; lazily
# populated.
_agents: dict[str, Any] = {}
_agent_modules: dict[str, Any] = {}  # name → imported module (for tool list)

_HERE = Path(__file__).resolve().parent
_EXAMPLES_ROOT = _HERE.parent  # ``examples/``
_STATIC = _HERE / "static"


def _discover_agents() -> dict[str, Path]:
    """Return {agent_name: path-to-agent.py} for every example exposing
    ``build_client``. Skips the playground itself.
    """
    found: dict[str, Path] = {}
    for project_dir in sorted(_EXAMPLES_ROOT.iterdir()):
        if not project_dir.is_dir() or project_dir.name == "playground":
            continue
        agent_py = project_dir / "agent.py"
        if not agent_py.is_file():
            continue
        # Cheap check: grep for "def build_client" before doing the
        # expensive import. Avoids importing every example on startup.
        if "def build_client" in agent_py.read_text(encoding="utf-8"):
            found[project_dir.name] = agent_py
    return found


def _load_agent(name: str) -> Any:
    """Lazy-load and cache an agent by name."""
    if name in _agents:
        return _agents[name]
    discovered = _discover_agents()
    if name not in discovered:
        raise HTTPException(404, f"agent {name!r} not found")
    agent_py = discovered[name]
    spec = importlib.util.spec_from_file_location(f"_playground_agents.{name}", agent_py)
    assert spec is not None and spec.loader is not None
    module = importlib.util.module_from_spec(spec)
    sys.modules[spec.name] = module
    # Adding the agent's directory to sys.path so its sibling imports
    # (e.g. ``from tools import ...``) resolve. We never remove the
    # path — the playground process is short-lived.
    sys.path.insert(0, str(agent_py.parent))
    try:
        spec.loader.exec_module(module)
    except Exception as e:
        raise HTTPException(500, f"failed to import agent {name!r}: {e}\n{traceback.format_exc()}")
    if not hasattr(module, "build_client"):
        raise HTTPException(500, f"agent {name!r} has no ``build_client`` function")
    try:
        client = module.build_client()
    except Exception as e:
        raise HTTPException(500, f"build_client() raised: {e}\n{traceback.format_exc()}")

    # Attach PlaygroundTraceMiddleware to the front of the chain so it
    # observes every request without modifying it. We do this by
    # reaching into the client's middleware pipeline if accessible;
    # if not, document that traces are best-effort.
    try:
        from tracing import PlaygroundTraceMiddleware
        # Try common pipeline access patterns; adapt based on actual SDK.
        if hasattr(client, "_pipeline") and hasattr(client._pipeline, "_middlewares"):
            client._pipeline._middlewares.insert(0, PlaygroundTraceMiddleware())
    except Exception:
        pass  # tracing is best-effort

    _agents[name] = client
    _agent_modules[name] = module
    return client


# ---- API models ----------------------------------------------------------


class AgentInfo(BaseModel):
    name: str
    description: str = ""
    tool_names: list[str] = []


class ChatRequest(BaseModel):
    message: str


class ChatResponse(BaseModel):
    text: str
    trace: list[dict] = []


class ToolInvocation(BaseModel):
    arguments: dict[str, Any] = {}


# ---- App + routes --------------------------------------------------------


app = FastAPI(title="EAP-Core Playground")


@app.get("/api/agents", response_model=list[AgentInfo])
async def list_agents() -> list[AgentInfo]:
    """List every example agent the playground discovered."""
    discovered = _discover_agents()
    result = []
    for name in discovered:
        try:
            client = _load_agent(name)
            tool_names = (
                sorted(t.name for t in client.tool_registry.list_tools())
                if hasattr(client, "tool_registry") and client.tool_registry
                else []
            )
        except Exception as e:
            # Surface load failures rather than hiding them.
            result.append(AgentInfo(name=name, description=f"(load error: {e})", tool_names=[]))
            continue
        description = (_agent_modules[name].__doc__ or "").strip().split("\n")[0]
        result.append(AgentInfo(name=name, description=description, tool_names=tool_names))
    return result


@app.post("/api/agents/{name}/chat", response_model=ChatResponse)
async def chat(name: str, body: ChatRequest) -> ChatResponse:
    """Send a message to the agent; return the agent's response + trace."""
    client = _load_agent(name)
    try:
        result = await client.generate_text(body.message)
    except Exception as e:
        raise HTTPException(500, f"generate_text raised: {e}")
    # generate_text returns either a Response object or a string,
    # depending on SDK version; normalise.
    text = result.text if hasattr(result, "text") else str(result)
    # Trace lives on the last Context the pipeline touched; the SDK
    # may or may not surface it back to the caller. For the playground
    # we capture via a thread-local on the middleware itself (simpler).
    trace: list[dict] = []
    if hasattr(client, "_last_trace"):
        trace = client._last_trace
    return ChatResponse(text=text, trace=trace)


@app.post("/api/agents/{name}/tools/{tool}")
async def invoke_tool(name: str, tool: str, body: ToolInvocation) -> dict:
    """Invoke a tool on the agent directly, bypassing the LLM."""
    client = _load_agent(name)
    if not hasattr(client, "tool_registry") or client.tool_registry is None:
        raise HTTPException(400, f"agent {name!r} has no tool registry")
    try:
        result = await client.tool_registry.invoke(tool, body.arguments)
    except Exception as e:
        raise HTTPException(500, f"tool {tool!r} raised: {e}")
    return {"result": result}


# ---- Static files (frontend) ---------------------------------------------


if _STATIC.is_dir():
    app.mount("/static", StaticFiles(directory=_STATIC), name="static")


@app.get("/")
async def index() -> FileResponse:
    return FileResponse(_STATIC / "index.html")


def main() -> None:
    """Entry point for `python server.py`."""
    import uvicorn
    uvicorn.run(app, host="127.0.0.1", port=8765, log_level="info")


if __name__ == "__main__":
    main()
```

Key design notes:
- **Discovery via grep before import.** Reading the file and checking for `"def build_client"` avoids importing all examples on startup. Imports happen lazily on first reference.
- **`sys.path` mutation per agent.** Each example has its own `tools/` or local imports; adding the agent's dir to `sys.path` lets those resolve. Side effect of a long-running playground: paths accumulate. Acceptable for an example.
- **Trace surfacing is fragile.** The middleware writes to `ctx.metadata`, but the playground server doesn't see the same `Context` instance after `generate_text` returns. Step 1.4 below adds a thread-local on the middleware so the playground can read the last trace.

The cleanest approach: `PlaygroundTraceMiddleware` writes to a contextvar / threading-local, and `server.py` reads from it after `generate_text`. Update `tracing.py`:

```python
import contextvars

_current_trace: contextvars.ContextVar[list[dict] | None] = contextvars.ContextVar(
    "playground_trace", default=None
)


class PlaygroundTraceMiddleware(PassthroughMiddleware):
    name = "playground_trace"

    async def on_request(self, req: Request, ctx: Context) -> Request:
        trace: list[dict] = []
        _current_trace.set(trace)
        ctx.metadata["playground.trace"] = trace
        ctx.metadata["playground.trace_start"] = time.perf_counter()
        return req

    # ... rest as before
```

Then in `server.py`, after `generate_text`:

```python
from tracing import _current_trace
trace = _current_trace.get() or []
```

ContextVars are async-safe; each `generate_text` call gets its own trace.

```python
def _wrap_invoke_for_trace(client) -> None:
    if not hasattr(client, "tool_registry") or client.tool_registry is None:
        return
    original = client.tool_registry.invoke
    async def traced(name, args):
        trace = _current_trace.get()
        start = time.perf_counter()
        try:
            result = await original(name, args)
            if trace is not None:
                trace.append({
                    "kind": "tool_call",
                    "name": name,
                    "args": _make_json_safe(args),
                    "result": _make_json_safe(result),
                    "duration_ms": (time.perf_counter() - start) * 1000,
                })
            return result
        except Exception as e:
            if trace is not None:
                trace.append({
                    "kind": "tool_error",
                    "name": name,
                    "args": _make_json_safe(args),
                    "error": str(e),
                    "duration_ms": (time.perf_counter() - start) * 1000,
                })
            raise
    client.tool_registry.invoke = traced
```

- [ ] **Step 2.2: `app.js`** — fetch agents on load, wire chat + tool forms

```js
// EAP-Core Playground frontend. No framework — vanilla DOM + fetch.

const agentSelect = document.getElementById("agent-select");
const toolList = document.getElementById("tool-list");
const toolSelect = document.getElementById("tool-select");
const messages = document.getElementById("messages");
const chatForm = document.getElementById("chat-form");
const chatInput = document.getElementById("chat-input");
const tracePanel = document.getElementById("trace-panel");
const traceList = document.getElementById("trace-list");
const toolForm = document.getElementById("tool-form");
const toolArgs = document.getElementById("tool-args");
const toolResult = document.getElementById("tool-result");

let agents = [];

async function loadAgents() {
  const resp = await fetch("/api/agents");
  agents = await resp.json();
  agentSelect.innerHTML = "";
  for (const a of agents) {
    const opt = document.createElement("option");
    opt.value = a.name;
    opt.textContent = `${a.name} — ${a.description || "(no description)"}`;
    agentSelect.appendChild(opt);
  }
  agentSelect.addEventListener("change", refreshSidebar);
  refreshSidebar();
}

function refreshSidebar() {
  const a = agents.find((x) => x.name === agentSelect.value);
  toolList.innerHTML = "";
  toolSelect.innerHTML = "";
  if (!a) return;
  for (const t of a.tool_names) {
    const li = document.createElement("li");
    li.textContent = t;
    toolList.appendChild(li);
    const opt = document.createElement("option");
    opt.value = t;
    opt.textContent = t;
    toolSelect.appendChild(opt);
  }
}

function appendMessage(role, text) {
  const div = document.createElement("div");
  div.className = `msg msg-${role}`;
  div.textContent = `${role}: ${text}`;
  messages.appendChild(div);
  messages.scrollTop = messages.scrollHeight;
}

function renderTrace(trace) {
  traceList.innerHTML = "";
  for (const entry of trace) {
    const li = document.createElement("li");
    li.textContent = `[${entry.ts_ms?.toFixed?.(1) ?? "—"}ms] ${entry.kind}: ${entry.name || ""} ${
      entry.args ? JSON.stringify(entry.args) : ""
    }`;
    traceList.appendChild(li);
  }
  tracePanel.querySelector("summary").textContent =
    `Tool-call trace (${trace.length} entries)`;
}

chatForm.addEventListener("submit", async (e) => {
  e.preventDefault();
  const msg = chatInput.value.trim();
  if (!msg) return;
  const agentName = agentSelect.value;
  if (!agentName) return alert("pick an agent first");
  appendMessage("user", msg);
  chatInput.value = "";
  chatInput.disabled = true;
  try {
    const resp = await fetch(`/api/agents/${agentName}/chat`, {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ message: msg }),
    });
    if (!resp.ok) {
      const err = await resp.text();
      appendMessage("error", `${resp.status}: ${err}`);
    } else {
      const data = await resp.json();
      appendMessage("agent", data.text);
      renderTrace(data.trace || []);
    }
  } finally {
    chatInput.disabled = false;
    chatInput.focus();
  }
});

toolForm.addEventListener("submit", async (e) => {
  e.preventDefault();
  const agentName = agentSelect.value;
  const tool = toolSelect.value;
  if (!agentName || !tool) return;
  let args = {};
  try {
    args = toolArgs.value.trim() ? JSON.parse(toolArgs.value) : {};
  } catch (err) {
    toolResult.textContent = `Invalid JSON: ${err.message}`;
    return;
  }
  toolResult.textContent = "(invoking…)";
  const resp = await fetch(`/api/agents/${agentName}/tools/${tool}`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ arguments: args }),
  });
  const data = await resp.json();
  toolResult.textContent = JSON.stringify(data, null, 2);
});

loadAgents();
```

```python
    data = resp.json()
    assert "text" in data
    assert "trace" in data
    assert isinstance(data["trace"], list)
```
